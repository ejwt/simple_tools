功能：从avi文件中提取出音频文件和视频文件。提取得到的音频文件的扩展名设置为.mp3，提取的视频文件的扩展名设置为.mp4u。提取音频文件是本程序的主要目的。

这是付兴在周阳“extract_avi_mp4_mp3”工程的基础上修改得到的。此程序只能适用于avi文件(AVI-2)中只有一路音频流和一路视频流的情况。开发环境是VC++ 6.0。

付兴的改动：

v2:
1. 添加更多的判断条件，当发现处理的文件不是avi文件，或者文件大小有问题时，给出提示，用户可以选择退出，也可以选择强行继续计算；提高对非标文件和非avi文件的适应性。
2. 修正bug: 在'movi' chunk中，如果紧接着'movi'的不是音频或者视频数据，而是索引，程序就无法提取出音频、视频数据，保存成新文件。实际上，这种情况只可能出现在OpenDML AVI File Format Extensions这些扩展了的avi文件的格式中，索引不仅仅出现在文件的末尾。
3. 修正bug: 程序可能死循环。修改方法：使得读取avi文件的指针在每次循环中至少向前移动4个字节，并判断当前处理的avi文件的位置是否到了文件末尾；这样，不论文件内容如何，都可以避免死循环。

v1:
1. 添加注释，整理代码，使得可读性更好。
2. 程序运行时可以选择是否保存提取出来的mp3文件和mp4u文件，如果不保存mp4u文件，可以减少大量磁盘写入操作，提高程序执行的速度。
